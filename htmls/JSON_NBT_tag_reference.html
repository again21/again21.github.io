JSON NBT标签参考

描述 标签类型
结束型(End) 
end
字节型(Byte) 
byte
短整型(Short) 
short
整型(Integer) 
int
长整型(Long) 
long
单精度浮点型(Float) 
float
双精度浮点型(Double) 
double
字节型数组(Byte Array) 
byteArray
字符串(String) 
string
列表型(List) 
list
复合型(Compound) 
compound
整型数组(Integer Array) 
intArray
长整型数组(Long Array)（仅Java版） 
longArray

NBT根部标签示例

{
    "name": "<根部标签的键（可以不填）>",
    "value": {
        <子标签>
    }
}

单个子标签示例

"<标签的键>": {
    "type": "<标签类型>",
    "value": <标签的值>
}

字节型、短整型、整型示例

123
-34

长整型示例（JavaScript不支持长整型，所以分成两个整型）

[114514,0]
[7,-3]

单精度浮点型、双精度浮点型示例

123.45
-1.23

字节型数组、整型数组示例

[11,45,14]
[-12,34]

字符串示例

"Hello World"

列表型示例

{
    "type": "<子标签类型>",
    "value": [
        <子标签的值>
    ]
}

值为空使用结束型填充

{
    "type": "end",
    "value": []
}

复合型示例

{
    <子标签>
}

长整型数组示例

[[114,0],[514,0]]
[[20,-5],[34,-2]]

完整示例

{
    "name": "",
    "value": {
        "aaa": {// 字节型
            "type": "byte",
            "value": 34
        },
        "bbb": {// 短整型
            "type": "short",
            "value": 520
        },
        "ccc": {// 整型
            "type": "int",
            "value": 123
        },
        "ddd": {// 长整型
            "type": "long",
            "value": [
                114514,
                0
            ]
        },
        "eee": {// 单精度浮点型
            "type": "float",
            "value": 12.3
        },
        "fff": {// 双精度浮点型
            "type": "double",
            "value": 45.6
        },
        "ggg": {// 字节型数组
            "type": "byteArray",
            "value": [
                11,
                45,
                14
            ]
        },
        "hhh": {// 字符串
            "type": "string",
            "value": "Hello World"
        },
        "iii": {// 列表型
            "type": "list",
            "value": {
                "type": "string",
                "value": [
                    "Hello",
                    "World"
                ]
            }
        },
        "jjj": {// 复合型
            "type": "compound",
            "value": {
                "t1": {
                    "type": "byte",
                    "value": 24
                },
                "t2": {
                    "type": "short",
                    "value": 234
                }
            }
        },
        "kkk": {// 整型数组
            "type": "intArray",
            "value": [
                1234,
                567,
                890
            ]
        },
        "lll": {// 长整型数组
            "type": "longArray",
            "value": [
                [
                    114,
                    0
                ],
                [
                    514,
                    0
                ],
                [
                    1919,
                    0
                ],
                [
                    810,
                    0
                ]
            ]
        }
    }
}
